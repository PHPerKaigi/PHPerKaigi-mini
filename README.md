## PHPerKaigi mini の概要

PHPerKaigi mini は技術カンファレンス PHPerKaigi のスポンサーさまおよびスポンサーをご検討頂いている企業さまにホストになって頂いてPHP関連技術者（PHPer / ペチパー）向けに開催される勉強会イベントです。

* 平日の19:00-22:00で開催する
* 1ヶ月に1回実施する
* 参加者募集はconnpassで実施する
* 以下のフォーマットで実施する
  * オープニングのタイミングで乾杯
  * オープニング後にホスト企業さまの会社紹介（5分）
  * 会社紹介後に参加者全員で自己紹介を実施する
  * 事前公募したLT（5分）を5本〜8本ほど実施
  * LTのテーマは「PHPerが聞いて面白ければOK」
  * LTのうち1〜2本はホスト企業さま枠として実施可
  * LT終了後、懇親会開始
* 録画・配信はホスト企業さまのご希望に応じて実施
  * 実施の場合、ライブ配信にPHPerKaigiのYouTubeチャンネルも利用可能

## ホスト企業さまへのお願い

PHPerKaigi mini 開催のホスト企業を募集しています。
過去に PHPerKaigi のスポンサーをして頂いた企業さま、次回以降開催のスポンサーをご検討頂いている企業さま、ご検討ください。

* 会場として部屋をご提供ください
  * おおむね20名〜名入れる部屋をご用意ください
  * 人数はご相談可能です
  * プロジェクターとマイクをご用意ください
  * 設営・撤収はホスト企業さまにてご実施ください
* アルコール含む飲み物（必須）・おつまみ（任意）・軽食（任意）をご提供ください
  * 注文・配布などはホスト企業さまにてご実施ください
* PHPerKaigi miniの開催について、ホスト企業さまからPHPerKaigi運営へのスポンサー費用などの支払いは発生しません

立候補頂ける企業さまは [これからのPHPerKaigi mini](https://github.com/PHPerKaigi/PHPerKaigi-mini/issues) ページからタイトルに *募集中* とあるもののコメント欄で @hasegawa-tomoki を入れる形で以下とともにコメントしてください。  

* 企業名
* 希望日
* connpassに追加する管理者のアカウント名
* お借りできるスペースのキャパシティ（人数）

## 開催までの流れ

### 開催2ヶ月〜3ヶ月前

3ヶ月前: 過去にPHPerKaigiにスポンサーご参加頂いているホスト企業さま優先 開催予約  
2ヶ月前: PHPerKaigiへのスポンサーをご検討頂いてるホスト企業さま優先 開催予約

* 開催の予約（日付・場所の決定）
* 人数の決定

### 開催1.5ヶ月前まで

* イベントページ（connpass）の作成
  * ホスト企業さまご担当者を管理者として追加
  * 入場方法などをご担当者さまにて記入

### 開催1ヶ月前

* イベントページを公開 & 参加者の募集開始
* fortee.jpにイベントページを作成

### 開催数日前

* fortee.jpにタイムテーブル（発表順）掲載
